3、ng-bind-template

官方api：The ngBindTemplate directive specifies that the element text content should be replaced with the interpolation of the template in the ngBindTemplate attribute. Unlike ngBind, the ngBindTemplate can contain multiple {{ }} expressions. This directive is needed since some HTML elements (such as TITLE and OPTION) cannot contain SPAN elements.
大概意思：ngBindTemplate指令指定的文本内容应该替换为插入的值（应该是$scope的属性），不想ng-bind，ngBindTemplate可以包含多个{{}}表达式；这个指令是必要的，因为一些元素不能包含span标签
